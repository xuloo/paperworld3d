Packagecom.paperworld.util.lang
Classpublic class LanguageManager
InheritanceLanguageManager flash.events.EventDispatcher

LanguageManager provides the ability to handle multiple languages.

Locale class is not flexible enough to support this cores needs - i.e. load multiple language definition files for one language.

Tests have shown that once a language file is loaded it will not be replaced with any subsequent loading of a different file for the same language.

The language definition files will be in the standard XLIFF format but all loading and string replacement / lookups will be handled by this package.

XML FILE MUST BE SAVED IN UNICODE FORMAT FOR SPECIAL CHARACTERS TO WORK. %0a Used as RETURN CARRIAGE, unescape() all language text.
