Packagecom.paperworld.preload.filerequests
Classpublic class FileRequest
InheritanceFileRequest flash.events.EventDispatcher
SubclassesFARFileRequest, LoaderFileRequest, URLLoaderFileRequest

FileRequest is an abstract base class for all the file requests that extend it.

It provides a set of properties and methods that can be overridden by children and a single Factory method getFileRequest that is used to return the correct FileRequest type for the file being loaded.
