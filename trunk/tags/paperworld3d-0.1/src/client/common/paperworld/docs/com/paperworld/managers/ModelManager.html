Class ModelManagerMethods
 
Packagecom.paperworld.managers
Classpublic class ModelManager

Holds keyed references to all the models listed in the configuration file supplied at runtime.

Models are held in Dictionary instances - each type of model (e.g. Collada, Ase etc) has its own Dictionary containing all the models of that type keyed by the name supplied in the configuration file.

public function addModel(key:String, type:String, value:EventDispatcher):void

Adds a model to the manager's list. Checks if a model of this type has already been supplied, if not creates a new Dictionary to store these types of models in.

Parameters
key:String — The name of the model being added. This name can be used to retrieve the model later.
 
type:String — The type of model being added (e.g. Collada, Ase, MD2 etc).
 
value:EventDispatcher — The loader object that was used to load the model. The model's data will be available from this object once the load has completed.
