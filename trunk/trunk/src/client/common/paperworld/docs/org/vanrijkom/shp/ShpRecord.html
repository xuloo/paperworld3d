Packageorg.vanrijkom.shp
Classpublic class ShpRecord

The ShpPoint class parses an ESRI Shapefile Record Header from a ByteArray as well as its associated Shape Object. The parsed object is stored as a ShpObject that can be cast to a specialized ShpObject deriving class using the found shapeType value.
