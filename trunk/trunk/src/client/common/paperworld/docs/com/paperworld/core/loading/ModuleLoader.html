Classpublic class ModuleLoader
InheritanceModuleLoader flash.events.EventDispatcher

The ModuleLoader class provides the functionality to load all the components of a module and any required modules defined by the modules configuration. It also provides the option to use the application's default loading skin and behaviour to provide a graphical interface for the loading process.

public function load(event:Event = null):void

Start the module loading process. Get the ModuleData object that holds the properties for the module that's been requested. Creates a new LoaderQueue and sets up its listeners. Calls addComponents() to build the load queue and then starts the queue loading.

Parameters
event:Event (default = null) — Optional parameter, triggers the object to commence loading of all the components requested.
