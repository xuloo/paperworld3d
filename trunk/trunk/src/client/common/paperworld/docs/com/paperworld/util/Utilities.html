getClassInstance()method 
public static function getClassInstance(key:String):*

Utility method for creating an instance of a class via AS3's Reflection function.

You must ensure you cast the object returned by this method, though, to the correct type (generally Bitmap, MovieClip or Sound) in order to use it, it saves an inline function call at the expense of the time it takes to make this extra function call. As these calls are generally made at initialisation time rather than during the course of a game your user isn't really going to notice the delay.

Parameters
key:String — the String name of the Class instance you want to be returned.

Returns
* — Returns an instance of the class requested, type is not specified but will generally be either Bitmap, MovieClip or Sound, remember to cast to the appropriate type when calling this method.
