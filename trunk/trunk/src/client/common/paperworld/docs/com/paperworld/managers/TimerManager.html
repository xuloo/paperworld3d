Packagecom.paperworld.managers
Classpublic class TimerManager

TimerManager provides access to all Timer events that occur within the application.

By keeping all Timers in a central location they can be accessed from anywhere and efficiently killed from one call. This helps make Garbage Collection a little easier and prevents you from forgetting about a Timer and keeping it running without knowing causing memory leakage and other annoyances.
